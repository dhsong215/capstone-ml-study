## 빠른 요약

- **데이터**: 네이버 DataLab API (무료) + 썸트렌드 1개월 (18만원)
- **키워드**: 과자/베이커리 Top 500 (2022~현재, 4년)
- **모델**: XGBoost Regressor + LSTM Regressor + Logistic Regression
- **타겟**: virality_score (연속값, Z-score 기반)
- **발표 킬러**: Top-10 Precision + SHAP 설명 + ML vs DL 비교
